Added a page on the [Open Knowledge Format (OKF) bundle](/spec/agent-readiness/okf-bundle/), a convention for packaging a whole knowledge base as a tree of typed Markdown concept files an agent can ingest in one fetch. The site now ships it as a worked example: the spec is generated into a browsable bundle at `/okf/` and a single `/okf.tar.gz`, advertised through our [AI Catalog](/spec/agent-readiness/agentic-resource-discovery/) — which now lists the OKF bundle and the Agent Skill alongside the MCP server and A2A agent — and through `/llms.txt`.
