Removed the pages on entry and exit animations (`@starting-style`, `transition-behavior`) and content-based field sizing (`field-sizing: content`). Both described CSS features whose benefit is authoring convenience rather than an auditable property of a good website — the spec describes outcomes for visitors, crawlers, and agents, not ways of building. The same test now gates new proposals.
